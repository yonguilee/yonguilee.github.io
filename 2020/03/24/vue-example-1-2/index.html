Vue로 UI 개발하기 - 좋아요 컴포넌트(튜토리얼1)

‘좋아요 컴포넌트’는 자주 볼 수 있는 UI 중 하나이다. 한번 쯤은 작업 flow를 끊어서 생각해보면 좋을 것 같아 컴포넌트 제작 과정을 상세하게 정리해보고자 한다. 이 글에서는 기본 Vue 탬플릿 구조와 사용된 vue 문법, 예제 구조에 대해 간략하게 설명한다.

컴포넌트 작업을 할때 <template> 하위에 하나의 태그로 묶여있어야 한다.

사용할 Vue 문법

  • 이중 중괄호
  • v-bind
  • v-for
  • v-show
  • v-on

작업 흐름

  • 기본 마크업
  • 기능 구현
    • 썸네일은 0 ~ 3개까지 노출된다.
    • 썸네일 3개 이상일 경우 ‘외 N명’ 노출된다.
    • 좋아요 버튼 토글 가능 + 숫자 업데이트가 되어야 한다.
