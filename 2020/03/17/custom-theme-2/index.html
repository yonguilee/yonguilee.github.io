헤더 영역

파비콘 뿐만 아니라 헤더 영역의 기본 설정을 수정 할 수 있다.

경로 : themes/icarus/_config.yml

디자인

key color랑 view를 좀 손보고 싶은 부분이 있는데, key color 부분의 css는 cdn으로 불러오는 것 같아서 조금 더 살펴 볼 예정..

검색 레이어 커스텀

  • 아이콘 변경

    fontawesome을 사용한다.

    개발자도구로 확인해보면 기본 닫기 아이콘은 fa-times-circle이고, 나는 fa-times로 수정할 것이다.

  • 디자인 수정

    내가 수정하려는 요소의 클래스를 검색해보니 다행히 검색 레이어에서만 사용되는 것 같다.

    블로그 전반에 사용되는 카드와의 느낌을 맞추기 위해 border-radius 를 추가하고, 여백등을 조정해 보겠다.

    .serachbox-container에 border-radius 속성을 추가해야되는데, 검색해보니 다른 페이지에서도 사용되는 것 같다. 그래서 검색 레이어에서만 쓰이는 insight.css에서 수정하였다.

    관련 CSS : themes/source/css/insight.css
