Github 블로그 만들기 with Hexo

One Note를 정리 노트로 사용했었는데, Github.io 블로그에 하나씩 옮겨보려 한다.
Github pages 와 Hexo로 블로그를 만드는 과정에 대한 글이다.


Github Pages 생성을 위한 저장소 생성

github에서 블로그용 프로젝트를 생성한다.

  • 저장소 명은 ‘유저명.github.io’ 로 생성하기
  • 로컬에 설치

유저명이 아닌 것으로 만들경우, https://yonguilee.github.io/yongui2.github.io 이런식으로 만들어질 것이다.


Hexo 설치

blog/ 하위 _config.yml에서 설정된 부분들을 개인에 맞게 커스텀 할 수 있다.


설정 파일 : _config.yml

_config.yml는 기본 설정 파일이다.
각 항목에 대한 자세한 설명은 Hexo Document > Configuration에서 확인 가능하다.

# Site

여기서 language는 테마의 기본 언어를 설정해주는 것으로, 아마.. 이 글을 읽는 사람은 모두 ko 라고 적겠지만, 다른 언어를 선택하고자 한다면 themes/languages에 있는 언어명 중에서 찾아 동일하게 적어주면 된다.

# Deployment

다음과 같이 블로그로 사용할 저장소를 추가해 주면 된다.


명령어

기본 페이지 생성

원하는 레이아웃(blog/scaffolds)으로 문서 source 파일(source/)을 생성한다.
디폴트 레이아웃은 post.md 이다.

로컬 빌드

로컬에서 돌려보면 4000번 포트로 확인 가능하다.

배포위한 페이지 생성

배포

간혹, 수정한 내용이 반영안되는 경우가 있는데 아래 명령어를 수행 후 다시 generate 후 deploy 하면 된다.


테마 설치 (icarus)

Hexo를 설치하면 기본테마는 landscape 이고, 마음에 들지 않을 경우 Hexo 테마에서 원하는 테마를 다운 받으면 된다.

icarus Git repo
icarus demo


테마 저장소 git clone

테마를 clone하게 되면 themes/ 하위에 icarus 폴더가 생성된다.

테마를 설치하면 icarus폴더 내에도 _config.yml 가 생성된다.

같은 설정파일이지만, 설정할 수 있는 부분이 다르니 잘 찾아서 활용하자.

blog/_config.yml : 블로그 설정 파일

blog/themes/icarus/_config.yml : 테마 관련 설정 파일


blog/_config.yml 에서 테마 변경하기


에러

Package cheerio is not installed

icarus 설치 후 , hexo server를 실행했을 때 마주할 수 있는 에러이다.

간단하게, cheerio를 설치해주면 된다.

cheerio 설치 후 hexo server를 다시 수행하면 잘 빌드된 화면을 볼 수 있다.

Deployer not found: git

배포를 위해 hexo deploy를 실행할 경우 마주할 수 있는 에러이다.

hexo 환경에서 github 배포를 위한 모듈을 설치해주면 된다.
